# Course Compass (MERN + AI + Redis)

## 🚀 Features

### 🔐 Authentication
- Admin Signup & Login
- Password hashing using bcrypt
- JWT-based authentication
- Protected routes

### 🤖 AI Recommendation
- Gemini AI integration (mocked)
- API: /api/recommendations

### 📚 Course Management
- Upload CSV → MongoDB
- Search courses
- Redis caching for performance

### ⚡ Frontend
- Next.js App Router
- React Query for caching
- Context API for state management
- Dynamic routing for course details

### 🐳 DevOps
- Dockerized backend
- CI/CD pipeline (GitHub Actions)
- Linux deployment with PM2 + Nginx

---

## 🛠️ Tech Stack

- Frontend: Next.js, React Query, Context API
- Backend: Node.js, Express
- Database: MongoDB
- Cache: Redis
- DevOps: Docker, GitHub Actions
